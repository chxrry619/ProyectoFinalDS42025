Acerca del Proyecto

Este sistema fue desarrollado como parte del proyecto final de la carrera de Data Science en la Universidad de Sonora (UNISON). El objetivo principal es ofrecer una plataforma interactiva para explorar revistas científicas, con especial énfasis en el análisis de su índice H (H-Index) y otros parámetros relevantes. A continuación, se describen las tecnologías y herramientas utilizadas en el desarrollo del sistema:

1. Tecnologías utilizadas en el backend

Python con Flask: Flask es un microframework web basado en Python que facilita la creación de aplicaciones web dinámicas. Se utilizó para manejar las rutas, la gestión de sesiones y la comunicación con las bases de datos, entre otras funcionalidades esenciales del sistema.

2. Diseño responsivo

Bootstrap 5: Bootstrap es un framework de diseño responsivo que permite crear interfaces de usuario adaptables a diferentes tamaños de pantalla. Gracias a esta herramienta, el sistema garantiza una experiencia óptima tanto en dispositivos móviles como en computadoras de escritorio.

3. Funcionalidades interactivas

DataTables: DataTables es una librería de JavaScript que se integra fácilmente con tablas HTML para proporcionar características avanzadas como la búsqueda, la paginación y el ordenamiento dinámico de los datos. En nuestro sistema, se utiliza para mostrar los datos de las revistas científicas de forma ordenada y fácil de explorar.

4. Extracción de datos desde la web

Beautiful Soup: Beautiful Soup es una librería de Python para el web scraping, utilizada para extraer información de páginas web. Esta herramienta nos permitió obtener datos de revistas científicas desde fuentes externas como SCImago y otros catálogos de revistas.

5. Fuentes de datos

Los datos utilizados en el sistema provienen de las siguientes fuentes:

  • SCImago Journal Rank (SJR): SJR es una base de datos que clasifica las revistas científicas a nivel mundial según su impacto y visibilidad. Utilizamos esta fuente para obtener el índice H (H-Index) de las revistas.
  • Catálogos institucionales de la UNISON: La Universidad de Sonora ofrece catálogos académicos que contienen información relevante sobre revistas científicas relacionadas con sus programas de investigación.
6. Propósito del sistema

El propósito de este sistema es proporcionar a los investigadores, estudiantes y académicos una herramienta accesible y fácil de usar para explorar revistas científicas y obtener información detallada sobre su impacto y visibilidad. El sistema permite la consulta de revistas en diversas áreas científicas y facilita la comparación de indicadores como el índice H, lo que puede ser útil para quienes buscan información actualizada sobre publicaciones científicas.
