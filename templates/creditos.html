Acerca del Proyecto Sistema desarrollado como parte del proyecto final de la carrera de Data Science en la UNISON. Su objetivo es explorar revistas científicas con énfasis en el H-Index y otros parámetros relevantes.
Tecnologías Backend Python con Flask: microframework para manejo de rutas, sesiones y lógica del servidor.
Diseño Responsivo Bootstrap 5: framework CSS para crear interfaces adaptables a distintos dispositivos.
Interactividad DataTables: librería para búsqueda, ordenamiento y paginación de tablas HTML.
Web Scraping Beautiful Soup 4.9.3: librería de Python para extraer datos estructurados desde sitios como SCImago.
Fuentes de Datos SCImago Journal Rank (SJR): datos de impacto de revistas.
Catálogos UNISON: información institucional sobre revistas científicas.
Propósito Ofrecer a investigadores y estudiantes una herramienta para explorar revistas científicas y comparar su impacto a través del H-Index.
Inteligencias Artificiales usadas ChatGPT y Github Copilot
