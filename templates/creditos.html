Acerca del Proyecto

Este sistema fue desarrollado como proyecto final utilizando las siguientes tecnologías:

  • Python con Flask para el backend
  • Bootstrap 5 para el diseño responsivo
  • DataTables para tablas interactivas
  • Beautiful Soup para web scraping

Los datos de las revistas fueron obtenidos de:

  • SCImago Journal Rank (SJR)
  • Catálogos institucionales de la UNISON
